컴퓨터공학기초실험
=============================
-FPGA 디자인 소프트웨어인 Quartus를 이용하여 add,sub 같은 산술연산, AND,OR의 논리연산부터 설계

-최종 프로젝트는 multiplier 기능이 추가된 ALU와 데이터를 주고 받는 BUS, 데이터를 저장하는 RAM을 설계

-testbench를 기반으로 총 13가지의 ALU 연산 기능을 test
